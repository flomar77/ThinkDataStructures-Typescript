# ThinkDataStructures-Typescript

An attempt to learn data structures using Typescript by using Allen B. Downey's Think Data Structures's book.

This is a TypeScript rewrite of the Java examples from Allen B. Downey's Think Data Structures. The original book and code can be found at [https://github.com/AllenDowney/ThinkDataStructures](https://github.com/AllenDowney/ThinkDataStructures).
